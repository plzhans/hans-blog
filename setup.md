
# 환경 구축

## 개발 환경

nvm 설치
```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

# 설치후 sheel 재시작
# source ~/.zshrc
# source ~/.bashrc

```

최신 lts 설치
```
nvm install --lts
```

설치 버전 확인
```
nvm list
```

사용 버전 선택
```
nvm use 24.13
```

참고) 버전 삭제
```
nvm uninstall 24.13
```
