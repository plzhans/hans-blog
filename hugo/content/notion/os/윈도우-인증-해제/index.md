윈도우 인증 해제


윈도우의 시리얼 인증을 잘못 진행해서 인증이 필요할 때 사용


1. 관리자 모드로 터미널 실행


```plain text
시작 -> 실행 -> cmd (관리자 모드로 실행)
```


2. 레지스트리 삭제


```plain text
slmgr /cpky
```


3. 윈도우 현재키 삭제


```plain text
slmgr /upk
```


4. 사전에 정품인증 클랙을 한 경우에는 해당 프로그램 삭제가 필요함.


아래 메뉴에 들어가서 KMS 등등 Acitve 등등이 바라보고 있는 크랙 프로그램 삭제 후에


일정도 같이 삭제 해줌
